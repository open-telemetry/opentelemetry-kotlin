# semconv-end-user-client

This module provides Kotlin representations of the semantic conventions defined in the federated
[End User Client semantic conventions](https://github.com/bidetofevil/semantic-conventions-end-user-client)
registry. Representations are only generated for conventions defined in that registry. Use the
constants generated by the `semconv` module for core OTel conventions.

Generated classes live in the `io.opentelemetry.kotlin.semconv.enduserclient` package so they cannot
clash with core classes that might have the same name (e.g. `AppAttributes`). Generation uses the shared weaver
templates in `semconv/templates`, parameterized by package.

This workflow can be invoked by running `./gradlew :semconv-end-user-client:generateSemanticConventions`.
